# jenkins

docker打包的jenkins环境，主要增加了常用插件。

# 运行方式

同[jenkins](https://github.com/cloudbees/jenkins-ci.org-docker)

# 包括插件

 * hipchat.hpi
 * greenballs.hpi
 * credentials.hpi
 * ssh-credentials.hpi
 * ssh-agent.hpi
 * git-client.hpi
 * git.hpi
 * github.hpi
 * github-api.hpi
 * ghprb.hpi
 * github-oauth.hpi
 * scm-api.hpi
 * postbuild-task.hpi
 * swarm.hpi
 * envinject.hpi
 * parameterized-trigger.hpi
 * build-name-setter.hpi
 * publish-over-ssh.hpi
 * deploy.hpi
